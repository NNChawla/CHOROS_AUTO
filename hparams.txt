The study is fully resumable — just rerun the same command:                                                                                                                                                                                                                                                                  
                                         
  conda run -n CHOROS python training/hparam_search.py --gpu 0 --n_trials 40                                                                                                                                                                                                                                                   
   
  The script checks if existing == 0 before seeding, so re-running on an existing study automatically skips the seed step and continues from where it left off. After 40 trials + 12 seeded, the next run starts with 52 observations and a much sharper TPE model.                                                            
                                                            
  Between batteries, look at where you are:                                                                                                                                                                                                                                                                                    
                                                            
  conda run -n CHOROS python -c "                                                                                                                                                                                                                                                                                              
  import optuna                                             
  optuna.logging.set_verbosity(optuna.logging.WARNING)
  study = optuna.load_study(                                                                                                                                                                                                                                                                                                   
      study_name='vr_encoder_mae_hps',
      storage='sqlite:////srv/CHOROS_AUTO/outputs/hparam_search/optuna.db',                                                                                                                                                                                                                                                    
  )                                                                                                                                                                                                                                                                                                                            
  completed = [t for t in study.trials if t.state.name == 'COMPLETE']
  print(f'{len(completed)} complete trials')                                                                                                                                                                                                                                                                                   
  best = study.best_trial                                   
  print(f'Best: #{best.number}  score={best.value:.4f}')                                                                                                                                                                                                                                                                       
  for k,v in sorted(best.params.items()): print(f'  {k}={v}')                                                                                                                                                                                                                                                                  
  "

  After ~80–100 total trials, it's worth checking Optuna's parameter importance to decide whether to narrow the search space for the next battery:                                                                                                                                                                             
                                                            
  conda run -n CHOROS python -c "                                                                                                                                                                                                                                                                                              
  import optuna                                             
  study = optuna.load_study(
      study_name='vr_encoder_mae_hps',
      storage='sqlite:////srv/CHOROS_AUTO/outputs/hparam_search/optuna.db',
  )                                                                                                                                                                                                                                                                                                                            
  importance = optuna.importance.get_param_importances(study)
  for k, v in sorted(importance.items(), key=lambda x: -x[1]):                                                                                                                                                                                                                                                                 
      print(f'  {k:25s}: {v:.3f}')                                                                                                                                                                                                                                                                                             
  "
                                                                                                                                                                                                                                                                                                                               
  If some parameters show near-zero importance (e.g., ffn_dim or sampling_alpha consistently), you can fix them to their best-observed value and start a new study with a tighter space — new study name so the TPE model is re-fit to the narrower distributions.

  Single GPU — everything automatic:                                                                                                                                                                                                                                                                                 
  conda run -n CHOROS python training/hparam_search.py --gpu 0 --n_trials 40
  After all trials finish it automatically runs the 20-epoch final model and the test evaluation.                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                     
  Parallel GPUs — designate one worker to run the final phase:                                                                                                                                                                                                                                                       
  # Worker 1: runs hparam search only                                                                                                                                                                                                                                                                                
  conda run -n CHOROS python training/hparam_search.py --gpu 0 --n_trials 20 --skip_final &                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                     
  # Worker 2: runs hparam search, then Phase 2+3 when done  
  conda run -n CHOROS python training/hparam_search.py --gpu 1 --n_trials 20                                                                                                                                                                                                                                         
  The second worker waits for all 20 of its own trials, then pulls the global best trial from the shared SQLite DB and proceeds to Phase 2+3.
